# devDive Frontend Repository

## Dev Dive
**This is a Q&A platform with a built-in gpt-like bot, which provides all the means to search for interesting data and communicate with other people.**

## Used technologies:
- ES6
- React
- React Router
- Axios
- SASS
- Webpack
- create-react-app utility
  
## [Figma prototype](https://www.figma.com/design/SnUxg5dEgxLQJxPnrDedA0/devDive?node-id=0-1&t=ncqgxTUhaFyUDWH7-1)

## Set up with npm
  ### 1. Install node.js from the [official site](https://nodejs.org/en)

  ### 2. Clone repository
  `git clone https://github.com/LehaGrand2018/devdive.git
  `
  ### 3. In the project directory, run: 
  `npm start`

  ### 4. Open [http://localhost:3000](http://localhost:3000) to view app in your browser.
